# Bundled Himalaya CLI (Linux)

Unmodified release archives from [soywod/himalaya](https://github.com/soywod/himalaya) **v1.2.0**:

| File | Target |
|------|--------|
| `himalaya.x86_64-linux.tgz` | 64-bit x86 Linux |
| `himalaya.aarch64-linux.tgz` | 64-bit ARM Linux |

License: MIT — see upstream repository. Twinbox extracts the `himalaya` binary on first use into `$TWINBOX_STATE_ROOT/runtime/bin/himalaya` when no `himalaya` is found in `PATH` and no binary exists there yet.
