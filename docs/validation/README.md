# Validation Artifacts

This directory is reserved for instance-specific validation reports, diagrams, and redacted local studies.

Runtime human-context inputs such as calibration notes now belong under `runtime/context/`, not `docs/validation/`.

For public releases, treat this directory as local-only by default.

Keep the stable public surface in:

- `docs/architecture.md`
- `docs/openclaw-progressive-validation-plan.md`
- `docs/release/`
- `docs/specs/`
