# 文档配图（`docs/assets/images`）

本目录存放 Twinbox 文档用 **PNG / WebP** 等静态配图。

- 新增图片时使用简短、有序的**文件名**（如 `01-intro.png`），并在引用该图的 Markdown 中写清相对路径（例如 `../assets/images/01-intro.png`）。
- 推荐宽度 **≤1200px**，以控制仓库体积。
