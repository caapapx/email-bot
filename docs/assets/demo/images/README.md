## 从飞书导出

1. 在飞书文档中按插图顺序导出或截屏。
2. 将文件保存为下表中的**文件名**（勿改序号与 slug），放入本目录。
3. 推荐 **PNG** 或 **WebP**；宽度建议 **≤1200px**，以控制仓库体积。
